# PCease

PCease is a full-stack PC builder web app where users can browse components, build PCs, check part compatibility, and share their builds. It also includes a community forum for discussions and feedback.

**Live Demo:** https://pcease.vercel.app

---

## Features
- Browse PC components (CPU, GPU, RAM, storage, etc.)
- Drag-and-drop PC builder
- Automatic compatibility checks
- Save and share builds
- Build recommendations
- Community forum
- Light / dark mode
- User authentication (JWT)

---

## Tech Stack
- **Frontend:** React, Vite, CSS
- **Backend:** Express, MongoDB
- **Auth:** JWT
- **Deployment:** Vercel
